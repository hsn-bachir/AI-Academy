1. Python Basics

Variables

To start with, variables are space created in memory to be either filled by programmer or users and manipulated all along the code

Data types of varibles:

  • Integer: includes all whole numbers 23, -4, 7...
  • Float: include all integers and decimal numbers 23.2, -4.48, 5.0...
  • String: set of charcters, should be entered between "" "this is ML course"...
  • Boolean: can have 2 values only, true or false

Variables are initialized by entering its name followed by = then its value x = 23

Input and output

A way to assign a value to the variable is by leting the user enter it using the function input

x=input("enter a value: ")

On the other hand, inorder to display the value of the variable use function print

You can display set of variables in one print using format function, we added{}as a placeholder to each variable we want to add in itsspecificlocation, we wrote f at the beginning of the print function, we added the variables inside {}! Lists

Lists are ordered collections of items that can be of any data type, including strings.

It is written in this form: l = ["hello", 23, True, ...]

Note that the positions in the lists starts with 0, such that [0]="hello", l[1]=23...

Some functions used in lists are:

  • append(): adds an item to the end of the list
  • insert(): adds an item at a specific position in the list
  • remove(): removes an item from the list
  • len(): computes the length of the list

2. Conditions

Can Python Decide?

Let's say you have 2 groups of friends one is going to football match other to cinema.Both events will occur on the same day and at the same time, so you cannot go at both events together. So now you need to choose 1 of the groups, for that you need something to help in this decision! Luckily, you like one activity more than the other one!
So if you like football more you will go with group 1 else with group 2.

Conditional statements allow you to execute different code based on certain conditions using if, elif, and else keywords.

x = 10
if x > 5:
  print("x is greater than 5")
elif x == 5:
  print("x is equal to 5")
else:
  print("x is less than 5")
#prints x is greater than 5

If Condition

If statements are used when we only have 1 condition to check. If this condition is met, we do the needed work and move on to other statements. If this condition is not met, we continue with other statements without changing anything, as if the statements for the condition true did not exist.

gender = input("enter your gender: ")
if gender == "male":
  print("Hello, Mr.")

#if the user enters male then this sentence will be displayed, else nothing happens

Else Condition

If-else statements are used when we only have 1 condition to check but 2 outcomes. If the condition is true, we do the if code, and if the condition is false, we do the else code.

gender = input("enter your gender: ")
if gender == "male":
  print("Hello, Mr.")
else:
  print("Hello, Ms.")

#if the user enters male then "Hello, Mr." will be displayed, else "Hello, Ms." is displayed

Elif Condition

If-elif statements are used when we have more than 1 condition to check. Since we have more than 1 condition, we should check each condition until one of them is true, after that, we end the process.

gender = input("enter your gender: ")
if gender == "male":
print("Hello, Mr.")
elif gender == "female":
print("Hello, Ms.")
else:
print("Hello, unknown gender")

#if the user enters male then "Hello, Mr." will be displayed, if the user enters female then "Hello, Ms." will be displayed, else " "Hello, unknown gender" will be displayed

3. Loops

Loops in Python are used to repeat actions multiple times. The most common types of loops are for loops and while loops.

For loops

The for loop iterates over a sequence like range:

for i in range(5):
  print(i) # Outputs 0 to 4

For each loops

The for each loop iterates over lists:

l = ["hello", 23, True, ...]
for i in l:
  print(i)

# Outputs hello, 23, True

While loops

The while loop continues as long as a condition is True:

count = 0
while count < 5:
  print(count)
  count += 1 # Increment count

# Outputs 0 to 4

5. Libraries

Python has a vast ecosystem of libraries that extend its functionality. Libraries can be imported using the import statement. Here's how you can use the math library:

import math
print(math.sqrt(16)) # Outputs: 4.0

Other popular libraries include numpy for numerical calculations, pandas for data manipulation, and matplotlib for data visualization. In this course we will be using Pandas and Matplotlib

  • Importing Files with Pandas

    Pandas is a powerful library for data analysis in Python. You can easily import various file types, such as CSV, Excel, and JSON, using Pandas.
    To import a CSV file, use the read_csv() function:
    import pandas as pd
    data = pd.read_csv('filename.csv')
    print(data.head())
    This code reads the file filename.csv and displays the first five rows using head().

  • Diagnosing Data with Pandas

    Use the info() method to get a summary of the data, including data types and null values:
    data.info()
    To find out how many null values each column has, use the isnull() and sum() methods:
    data.isnull().sum()
    The describe() method provides a summary of statistics for numerical columns:
    data.describe()

  • Introduction to Matplotlib

    Matplotlibis a popular library for creating visualizations in Python. It works well with Pandas and helps in visualizing trends, patterns, and relationships within data.
    To create a simple line plot, first import Matplotlib:
    import matplotlib.pyplot as plt
    Now, let's create a basic line plot:
    plt.plot([1, 2, 3, 4], [10, 20, 25, 30])
    plt.title('Sample Line Plot')
    plt.xlabel('X-axis Label')
    plt.ylabel('Y-axis Label')
    plt.show()
    You can use Matplotlib to plot data directly from a Pandas DataFrame:
    data['column_name'].plot(kind='hist')
    plt.title('Histogram of Column')
    plt.show()
    A bar plot is useful for comparing categories. Here's an example:
    categories = ['A', 'B', 'C']
    values = [5, 7, 3]
    plt.bar(categories, values)
    plt.title('Bar Plot Example')
    plt.xlabel('Category')
    plt.ylabel('Values')
    plt.show()
